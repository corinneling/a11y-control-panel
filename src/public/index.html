Control Panel for Visual Accessibility

Change the styling of any webpage to make it more accessible

About This Bookmarklet

A minimalist control panel that allows users to change the aesthetics of a page to make it more accessible, such as increasing contrast or font size. The accessibility icon expands on click, and each icon has a tooltip instructing the user on its capabilities. View on GitHub

How to Use:

  • Make sure your bookmarks bar is enabled
  • Drag the link below to your bookmarks bar
  • Click the link in your bookmarks bar on any webpage
